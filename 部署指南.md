# GitHub Pages 部署快速指南

## 一、上传代码到 GitHub

### 方式 1: 使用 Git 命令行 (推荐)

打开 PowerShell，依次执行以下命令：

```powershell
# 进入项目目录
cd D:\Desktop\CivilWar

# 初始化 git 仓库
git init

# 添加所有文件到暂存区
git add .

# 提交文件
git commit -m "Initial commit"

# 关联远程仓库（请替换为你自己的仓库地址）
git remote add origin https://github.com/你的用户名/你的仓库名.git

# 推送到 GitHub
git push -u origin main
```

如果遇到权限错误，可能需要先配置 Git：
```powershell
git config --global user.name "你的名字"
git config --global user.email "你的邮箱@example.com"
```

### 方式 2: 使用 GitHub Desktop

1. 下载并安装 GitHub Desktop: https://desktop.github.com/
2. 打开 GitHub Desktop，选择 "Add an Existing Repository"
3. 选择 `D:\Desktop\CivilWar` 文件夹
4. 点击 "Publish repository" 上传到 GitHub

### 方式 3: 通过 GitHub 网页上传

1. 访问 https://github.com/new 创建新仓库
2. 在仓库页面点击 "uploading an existing file"
3. 将所有文件拖拽到网页上传
4. 填写提交信息并确认

---

## 二、启用 GitHub Pages

1. 打开你的 GitHub 仓库页面
2. 点击顶部的 **Settings** 选项卡
3. 在左侧边栏找到并点击 **Pages**
4. 在 "Source" 部分：
   - Branch: 选择 **main** (或 master)
   - Folder: 选择 **/(root)**
5. 点击 **Save** 保存

等待 1-2 分钟，页面会自动刷新并显示你的网站链接。

---

## 三、访问你的网站

GitHub 会生成一个专属 URL，格式为：
```
https://你的用户名.github.io/你的仓库名/
```

例如：
```
https://zhangsan.github.io/lol-stats/
```

直接在浏览器中打开这个链接即可使用你的电竞选手数据分析面板！

## 四、更新数据

当你需要添加新的比赛数据时：

### 方法 1: 使用 Git 命令行

```powershell
cd D:\Desktop\CivilWar

# 添加新的 JSON 文件
git add .

# 提交更改
git commit -m "添加新的比赛数据"

# 推送更新
git push
```

### 方法 2: 通过 GitHub 网页

1. 打开你的 GitHub 仓库
2. 点击 "Add file" → "Upload files"
3. 上传新的或修改后的文件
4. 填写提交信息并确认

GitHub Pages 会在几分钟内自动更新你的网站。

## 五、常见问题

### Q1: 页面显示空白或无法加载数据？
**A:** 这是因为浏览器的 CORS 安全策略。确保：
- 使用 GitHub Pages 部署后访问
- 或者本地使用 http-server 运行（不能直接双击打开 html 文件）

### Q2: 如何添加新的比赛数据？
**A:** 有两种方式：
1. **自动扫描**：直接将 JSON 文件放入 `data` 目录，程序会自动加载
2. **手动配置**（推荐）：编辑 `data/file-list.json` 文件：
```json
{
  "files": [
    "0311_200317",
    "0311_205642",
    "新文件名.json"
  ]
}
```

### Q3: 可以自定义域名吗？
**A:** 可以！在 GitHub Pages 设置中添加 CNAME 记录即可。

### Q4: 网站是公开的还是私有的？
**A:** GitHub Pages 只对公共仓库免费且公开。如果需要私有仓库的 Pages，需要 GitHub Pro 账号。
